About The Project

Project Overview

This tool is designed for those who want to design their own website. Users can upload website screenshots or Figma links to receive UX feedback and redesign suggestions.

Technical Stack

  • GPT-4o: For analyzing screenshots/Figma files and providing feedback
  • Figma API: For accessing design files directly
  • Frontend: HTML, CSS, JavaScript

Visibility of System Status

Match Between System and the Real World

The terminology used in your navigation menu might be too technical for average users.

Suggestion: Use more familiar language and concepts that match users' real-world expectations.

Aesthetic and Minimalist Design
